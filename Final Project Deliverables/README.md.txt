Instructions to run the application:

app.py : Main source code file
requirements.txt : consists of a list of packages along with their versions, necessary dependencies to run our Flask application

Two ways to open the application on web browser:

Way 1

Step 1: Download the UI folder to local computer
Step 2: Run pip install -r requirements.txt in terminal to install all the necessary dependencies require to run the file
Step 3: Run python app.py to run the Flask application
Step 4: Output will generate a link "http://127.0.0.1:5000/" (Starts the application on a local server)
Step 5: Copy and paste the link on browser to open the web application

Way 2 

Just copy and paste the public URL generated by ngrok " https://0d22-103-25-231-102.ngrok-free.app " on web browser (Here, exposing the application to internet using ngrok)
